### [The Finding of Isabel](https://github.com/kinglyvillamor/the-finding-of-isabel)

`Unity` · `C#` · `Blender` · `Mixamo` · `Unity Pro-Builder`

`Vue.js` · `JavaScript` · `HTML/CSS` · `Chart.js`
